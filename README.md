# Text Editor

## Description
This project is a simple text editor web application that allows users to create and edit text documents.

## Table of Contents

- [Features](#features)
- [Technologies Used](#technologies-used)
- [Installation](#installation)
- [Usage](#usage)
- [License](#license)

## Features
- Create new text documents
- Edit existing text documents
- Save text documents
- Load text documents
- Basic text editing functionalities (e.g., copy, paste, undo)

## Technologies Used
- HTML
- CSS
- JavaScript
- Express.js (for server-side code)
- Node.js (for server-side runtime environment)

## Installation
1. Clone the repository to your local machine: git@github.com:awatson622/text-editor.git
2. Navigate to the project directory
3. Install dependencies: npm install

## Usage
1. Start the server: npm run start
2. Open a web browser and navigate to `http://localhost:3000` to access the text editor application.
3. Use the text editor to create, edit, save, and load text documents.
4. Enjoy using the text editor!
